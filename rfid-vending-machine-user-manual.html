💳 How to Use the Machine

  1. Walk up to machine - Check napkin count on display
  2. Hold card near scanner - Place RFID card close to front panel
  3. Wait for message - Screen shows authorization status
  4. Take napkins - Machine dispenses automatically

Screen Messages

DisplayMeaning
✅ "Authorized"Success - napkins dispensed
❌ "Unauthorized"Card not registered
⚠️ "Limit Reached"Used daily allowance
🔴 "Out of Stock"Machine empty
💡 Daily Limit: Each person gets 3 napkins per day (resets at midnight). Can be modified by admin in web app.
📞 Need Help? Contact your system administrator or IT support

🌐 Admin Setup (For Administrators Only)

  1. Press WiFi button on back of machine
  2. Connect to "Lyra-Vending-AP" (Password: lyra12345)
  3. Open browser to: 192.168.4.1

Admin Functions

  • 📊 Dashboard: Stock, cards, transactions
  • 💳 Add Cards: Register employee RFID cards
  • 👥 Manage: Enable/disable cards
  • ⚙️ Settings: Daily limits, stock reset
  • 📊 Reports: Download usage data

Adding Employee Cards

  1. Click "RFID Cards" then "Add Card"
  2. Click "Scan Card" and tap card on machine
  3. Enter employee name and ID
  4. Click "Add Card" to save
💡 WiFi Auto-Off: WiFi turns off after 5 minutes to save power. Press button again to reactivate.

🔧 Troubleshooting

Common Issues & Quick Fixes

❌ "Card not detected"
• Hold card closer • Try different angle • Check for damage
❌ "Unauthorized"
• Card not registered - ask admin • Check if disabled
⚠️ "Daily Limit Reached"
• Used today's allowance • Try tomorrow • Ask admin to increase
🔴 "Out of Stock"
• Machine empty - tell maintenance • Press stock button after refill

Machine Not Working?

  1. Check power: Screen should display
  2. Restart: Unplug 10 sec, plug back in
  3. Test WiFi: Press WiFi button
  4. Get help: Contact IT support

Refilling Machine

🔄 Steps: 1. Open service panel 2. Load 25 packets 3. Press Stock Button 4. Verify reset
