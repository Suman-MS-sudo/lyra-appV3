  Routing notes:
  • Run a wide 5V power trace (≥1.5 mm) along the bottom edge.
  • Keep RFID module away from the relay (relay switching causes EMI).
  • Keep SPI traces (RFID) short and parallel — run on same PCB layer.
  • I2C bus (SDA/SCL) can be long — pull-ups keep signal integrity.
  • Add a 100 µF / 16 V electrolytic cap near the 5V input for relay
    inrush current.
  • Add a 100 nF ceramic decoupling cap at each IC's VCC pin.
